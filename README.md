# ShopReact - E-Commerce PWA

A responsive e-commerce Progressive Web App built with React.

## Features
- 📱 Fully responsive design
- 🛒 Product grid with add to cart
- 🌙 Dark navigation theme
- ⚙️ Service worker offline support
- 📦 Progressive Web App

## Getting Started

```bash
npm install
npm start
npm run build
npm run deploy
```

## Technologies
- React 19
- CSS Grid & Flexbox
- Service Workers
- PWA
